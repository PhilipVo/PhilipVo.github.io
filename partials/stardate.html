Stardate



Code on GitHub
About:

Stardate uses NASA's API to gather information of upcoming meteorites that are within viewing distance of Earth. This information is organized and displayed as events for registered users to discuss and possibly meetup to view the stars together. Users can also enter in an address for meetups, and the location will be pinned on a map for other users to view.

This site was built with a Python backend by using the Flask framework. MySQL was used as the database to store user and meteorite information. AJAX was used on the front end to allow users to asynchonously post messages and join events, and Bootstrap was used as the CSS framework.
