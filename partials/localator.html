Localator

Code on GitHub
About:

Localator uses iPhone's Core Location to determine the distance between connected users. Within a radius, Localator starts alerting users proportionately based on the distance between users. These alerts include a visual flashing of the screen, a beeping sound, and vibrations from the phone.

Localator was initialy built as an app that would allow users to easily find their friends in loud or dark environments, such as at concerts. But other applications for Localator has been found, such as locating people during disaster events.

Localator was built with Xcode 7 and Swift 2.3. A Node server is hosted to send user locations to other users through the use of sockets.
