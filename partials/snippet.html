Snippet



Code on GitHub
About:

Snippet was initially built as a programming text editor for iPhones/iPads. It includes a keyboard that allows for easier typing, and users can save their code to use later. There is also a chatroom for users to copy and send code to each other so that users may collaborate on code.

Snippet was built with Xcode 7 and Swift 2.3. Text files are saved on the device by using Core Data. A JavaScript Node server is running to serve sockets for the chatroom.
